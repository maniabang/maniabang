# 👋 안녕하세요, 이광훈(Adam KwangHoon Lee)입니다!

**실시간 데이터가 흐르는 화면**에서 사용자가 판단할 수 있게 만드는 프론트엔드 개발자입니다.  
WebSocket 시세·시계열 차트·대용량 테이블이 겹치는 화면도 읽히게 설계하고, **Next.js · TypeScript**로 체감 속도와 유지보수성을 함께 챙깁니다.
